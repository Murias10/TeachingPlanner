# Chapter 1: GENERAL DESCRIPTION

The School of Computer Engineering (EII) of the University of Oviedo manages its academic timetables through a legacy system that requires manual maintenance and lacks a proper administration interface. The goal of this project is to develop a web application that replaces the current system and provides the EII with a modern, accessible tool for managing academic schedules. The application allows authorised users to create and maintain academic calendars, manage degree programmes, subjects, groups, and classrooms, and handle timetable-related requests from teaching staff, while remaining compatible with the existing tools and data formats used within the school. The system supports three types of users: administrative staff with full control over the scheduling data, teaching staff who can consult their timetables and communicate changes, and the general public who can view the published schedules without authenticating.
