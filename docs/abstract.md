# Abstract

## TeachingPlanner: Sistema de Gestión de Horarios Académicos para la Escuela de Ingeniería Informática de la Universidad de Oviedo

La herramienta a construir servirá para automatizar el conjunto de tareas manuales que se tienen que realizar durante buena parte del curso académico relativas a: confección de horarios, reserva de aulas, gestión de eventos académicos, etc. Además, tendrá la capacidad de procesar la información generada por otras herramientas, como el sistema académico SIES y generar también información para dichos sistemas. De cara a sus usuarios, la herramienta se presentará como una aplicación web para la cual se podrán definir diferentes perfiles con distinto conjunto de privilegios.

La herramienta sustituye al sistema heredado de la Escuela de Ingeniería Informática, basado en la edición manual de ficheros de texto plano en el servidor. Incorpora detección automática de conflictos en tiempo real, un sistema de solicitudes de cambio integrado que reemplaza la gestión por correo electrónico, sincronización con Google Calendar y compatibilidad completa con el formato de ficheros del sistema legado para garantizar la interoperabilidad con otras aplicaciones del ecosistema de la escuela. El sistema también ofrece consulta pública de horarios sin autenticación.

## TeachingPlanner: Academic Schedule Management System for the School of Computer Engineering at the University of Oviedo

The tool to be built will serve to automate the set of manual tasks that have to be performed during a large part of the academic year related to: preparation of timetables, reservation of classrooms, management of academic events, etc. In addition, it will have the capacity to process the information generated by other tools, such as the SIES academic system, and also generate information for these systems. For its users, the tool will be presented as a web application for which different profiles with different sets of privileges can be defined.

The tool replaces the legacy system at the School of Computer Engineering, which was based on manual editing of plain-text files on the server. It incorporates automatic real-time conflict detection, an integrated change-request system that replaces the email-based workflow, Google Calendar synchronisation, and full compatibility with the legacy file format to ensure interoperability with other applications in the school's ecosystem. The system also provides public timetable access without authentication.
